# dotfiles
My dot files


`cargo install bat`  alternative to cat
`cargo install du-dust`   Alternative to du. Use `dust ~/Downloads`
`cargo install fd-find`   Alternative to find. Use: `fd controller`
`cargo install ripgrep`   Alternative to grep. Use: `rg TODO src`
`cargo install procs`     Alternative to ps. Use: `procs`
`cargo install tlrc`      Alternative to man. Use: `tldr ls`
`cargo install zoxide`    Alternative to cd. Use: `z ~/Downloads`
`cargo install lsd`       Alternative to ls.
